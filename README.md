## 🚀 Featured Project

### Linkcore — Digital Wallet & Remittance Platform

A full-stack fintech application built from scratch with a Spring Boot REST API, customer-facing web frontend, multi-currency transfer flow, wallet ledger, transaction history, and printable receipts.

**Backend** · [github.com/abdulkhadir07/digital-wallet-api](https://github.com/abdulkhadir07/digital-wallet-api)
- RESTful API with JWT authentication, protected endpoints, and an append-only wallet transaction ledger
- Multi-currency FX conversion via ExchangeRate-API with domestic, cross-border, and FX transfer pricing rules
- KYC-aware transfer validation for higher-value transfers
- Deposit and withdrawal framework with payment-method support designed for future provider integrations
- Stack: Java 17 · Spring Boot 4 · PostgreSQL · Spring Security · JWT · Docker · Render
- Live API: [digital-wallet-api-551y.onrender.com/swagger-ui/index.html](https://digital-wallet-api-551y.onrender.com/swagger-ui/index.html)

**Frontend** · [github.com/abdulkhadir07/wallet-frontend](https://github.com/abdulkhadir07/wallet-frontend)
- Multi-step registration with OTP verification, login, dashboard, live transfer quote preview, recipient search, activity history filters, transaction details, printable receipts, and profile management
- Stack: HTML · Tailwind CSS · Vanilla JavaScript (ES Modules) · Netlify
- Live app: [linkcoreweb.netlify.app](https://linkcoreweb.netlify.app)
